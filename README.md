# Harry-Potter-Playlist-Manager
Harry Potter Movie Playlist Manager is a C++ console-based application designed to manage and navigate a playlist of Harry Potter movies using a Doubly Linked List. Each movie is stored as a node containing details such as the movie title and release year.

The system allows users to:

Add new movies

Delete movies by title

Display the full playlist

Navigate forward and backward between movies

View the currently playing movie

This project demonstrates dynamic memory management, traversal, and pointer manipulationâ€”core concepts of data structures. It serves as a practical and interactive example of implementing a doubly linked list for real-world playlist management.
